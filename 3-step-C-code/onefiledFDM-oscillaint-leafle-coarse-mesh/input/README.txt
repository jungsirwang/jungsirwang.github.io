coor0.txt:  coordinates for the background mesh
elem0.txt:  connections for the background mesh

coor1.txt:  coordinates for the solid mesh
elem1.txt:  connections for the solid mesh


NOTE: the last column in 'elem0.txt' or 'elem1.txt'
      tells you the material number of the element/cell.
	  This means every element/cell can use diferent material properites.
	  The material properties are provided in file 'materials'

The meanings of other files are self explained inside the file.
